# Anxiety?
Anxiety lets you reset all app permissions with a single click. Don't hesitate to use it when you're suspicious about app permissions.
# Overview
Anxiety is a lightweight SwiftUI application that lives in your menu bar and provides a quick way to reset all macOS privacy permissions (TCC - Transparency, Consent, and Control) across all apps. It's lightweight, customizable, and works seamlessly on modern macOS versions.
